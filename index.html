Python Web 程式設計入門實戰

Posted on August 28, 2013 at 10:00 PM


課程介紹

Python 是一種直譯式物件導向程式語言，具備簡潔、易於學習和功能強大等特性，在 Google、Mozilla 、Youtube、Instangram 、Dropbox 等知名科技公司都大量使用 Python 當做開發語言。Python 可以應用的範圍非常廣泛，舉凡：資料科學（機器學習、人工智慧、自然語言處理）、網站開發、網路爬蟲、物聯網、嵌入式系統和遊戲設計等都可以看到 Python 的身影。此外，由於 Python 可以很容易地和 C/C++ 和 Java 等語言整合，所以又稱為膠水語言（glue language）。


不管你是想要開始學習寫程式或是轉職網站工程師提高自己工作的專業能力，又或是單純對於學習資訊技術有興趣。趕快加入我們，一起跟著教學經驗豐富的講師、助教還有同學們一起開始從零開始學 Python 網站程式開發！ 在學習完課程後學員除了對於 Python 網站程式開發有系統性的認識，更可以使用 Python 開發一個屬於自己的網站應用程式，為自己的履歷增添一筆。

上課環境

（若學員要使用自己NB）

  • Windows/Linux/Mac OS 作業系統
  • 文字編輯器
  • Chrome 或 Firefox 瀏覽器
  • Terminal 終端機（Windows 使用 cygwin 或 cmder）
  • Python 3 程式語言
  • Anaconda 開發環境

課程大綱

(整堂課程時間估計約320分鐘)

  • Python 程式設計初體驗
  • Python 開發環境建置（終端機與編輯器）與 Github 操作入門
  • Python 程式設計入門上（基本語法與流程控制）
  • Python 程式設計入門中（資料結構與函式）
  • Python 程式設計入門下（檔案處理與資料庫）
  • Python 套件管理基礎
  • HTML/CSS/JavaScript 簡易基礎
  • Python Web 初體驗與網際網路基礎
